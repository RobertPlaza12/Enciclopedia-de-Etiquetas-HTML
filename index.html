Etiquetas HTML

# Etiqueta Descripción Ejemplo de uso Extras
1 <table> Crea una tabla como esta que vemos <table>...</table> Usa <tr>, <th>, <td>
2 <h1>...<h6> Titulo en diferentes niveles <h2>Subtítulo</h2> SEO importante
3 <a> Define un hipervínculo <a href="https://ejemplo.com" target="_blank">Ir</a> Atributos: href, target, rel
4 <meta> Define metadatos <meta charset="UTF-8"> Muy usado para SEO
5 <body> Contenido visible <body><p>Hola</p></body> Todo lo que se muestra en la página
6 <ul> Lista no ordenada <ul><li>Item</li></ul> Se representa con viñetas
7 <form> Formulario <form action="/" method="post">...</form> Usa inputs, select, textarea
8 <span> Contenedor en línea <p>Texto <span style="color:red">rojo</span></p> Útil para aplicar estilos
9 <footer> Pie de página <footer>© 2025</footer> Elemento semántico
10 <img> Inserta una imagen <img src="foto.jpg" alt="Foto"> Atributos: src, alt, width, height
11 <nav> Sección de navegación <nav><a href="#">Inicio</a></nav> Usado para menús
12 <video> Inserta un video <video controls><source src="video.mp4" type="video/mp4"></video> Soporta subtítulos
13 <button> Botón interactivo <button type="submit">Enviar</button> Tipos: button, submit, reset
14 <title> Título de la página <title>Mi Web</title> Aparece en la pestaña del navegador
15 <input> Campo de formulario <input type="text" name="nombre"> Tipos: text, password, email...
16 <section> Sección de contenido <section><h2>Tema</h2></section> Elemento semántico
17 <div> Contenedor genérico <div class="caja">Contenido</div> Muy usado para maquetación
18 <head> Encabezado de página o sección <head><h1>Inicio</h1></head> Elemento semántico
19 <strong> Texto importante <p>Esto es <strong>clave</strong></p> Diferente de <b> (semántica)
20 <ol> Lista ordenada <ol><li>Uno</li></ol> Se numera automáticamente
21 <article> Contenido independiente y autónomo <article><h2>Noticia</h2></article> Elemento semántico
22 <aside> Contenido lateral o secundario <aside>Publicidad</aside> Usado para barras laterales
23 <br> Salto de línea Línea 1<br>Línea 2 No necesita etiqueta de cierre
24 <hr> Línea divisoria horizontal <hr> Separador visual
25 <iframe> Incrusta otra página <iframe src="pagina.html"></iframe> Usado para mapas o videos externos
26 <audio> Reproduce audio <audio controls><source src="sonido.mp3" type="audio/mpeg"></audio> Soporta varios formatos
27 <canvas> Área de dibujo dinámico <canvas id="miCanvas"></canvas> Manipulable con JavaScript
28 <svg> Gráficos vectoriales escalables <svg><circle cx="50" cy="50" r="40"/></svg> Escalable sin perder calidad
29 <abbr> Abreviatura <abbr title="Cascading Style Sheets">CSS</abbr> Muestra tooltip al pasar el mouse
30 <cite> Cita de una obra <cite>Don Quijote</cite> Generalmente en cursiva
31 <mark> Texto resaltado <mark>Importante</mark> Por defecto se ve amarillo
32 <code> Muestra código <code>console.log("Hola")</code> Fuente monoespaciada
33 <pre> Texto preformateado <pre>Linea 1 Linea 2</pre> Conserva saltos y espacios
34 <blockquote> Cita en bloque <blockquote>Texto citado</blockquote> Generalmente con sangría
35 <q> Cita en línea <q>Frase célebre</q> Añade comillas automáticamente
36 <sup> Superíndice H<sup>2</sup>O Usado en fórmulas
37 <sub> Subíndice CO<sub>2</sub> Usado en química y matemáticas
38 <details> Sección desplegable <details><summary>Ver más</summary>Texto oculto</details> Interactividad sin JS
39 <summary> Título del <details> <summary>Información</summary> Sirve como encabezado
40 <progress> Barra de progreso <progress value="60" max="100"></progress> Indica porcentaje completado
41 <time> Representa una fecha u hora <time datetime="2025-09-13">13 de septiembre 2025</time> Puede usarse en microdatos
42 <address> Información de contacto <address>contacto@ejemplo.com</address> Elemento semántico
43 <label> Etiqueta para un control de formulario <label for="nombre">Nombre:</label> Se asocia con <input>
44 <textarea> Campo de texto multilínea <textarea rows="4" cols="30"></textarea> Usado en formularios
45 <select> Lista desplegable <select><option>Opción</option></select> Usa <option> dentro
46 <option> Elemento de una lista desplegable <option value="1">Uno</option> Se coloca dentro de <select>
47 <optgroup> Agrupa opciones en un select <optgroup label="Grupo 1"><option>Opción</option></optgroup> Mejora accesibilidad
48 <fieldset> Agrupa controles en un formulario <fieldset><legend>Datos</legend></fieldset> Visualmente añade un recuadro
49 <legend> Título de un <fieldset> <legend>Información Personal</legend> Asociado a formularios
50 <datalist> Lista de sugerencias para inputs <input list="frutas"><datalist id="frutas"><option value="Manzana"></datalist> Funciona con <input list>
51 <output> Resultado de un cálculo <output name="resultado">42</output> Usado en formularios dinámicos
52 <main> Contenido principal del documento <main><h1>Blog</h1></main> Elemento semántico
53 <figure> Figura con su descripción <figure><img src="foto.jpg"><figcaption>Descripción</figcaption></figure> Usado con imágenes
54 <figcaption> Título o descripción de una figura <figcaption>Paisaje al atardecer</figcaption> Se coloca dentro de <figure>
55 <small> Texto en tamaño reducido <small>Nota al pie</small> Usado en disclaimers
56 <del> Texto eliminado <del>Precio viejo</del> Muestra tachado
57 <ins> Texto insertado <ins>Precio nuevo</ins> Suele subrayarse
58 <kbd> Entrada de teclado <kbd>Ctrl + C</kbd> Fuente monoespaciada
59 <samp> Salida de un programa <samp>Error 404</samp> Simula respuesta de consola
60 <var> Representa una variable <var>x</var> = 5 Usado en matemáticas y programación
61 <em> Texto enfatizado <em>Muy importante</em> Generalmente en cursiva
62 <i> Texto en cursiva <i>Texto inclinado</i> Presentacional, no semántico
63 <b> Texto en negrita <b>Resaltado</b> Presentacional, diferente de <strong>
64 <u> Texto subrayado <u>Subrayado</u> Uso visual, no semántico
65 <s> Texto tachado <s>Texto obsoleto</s> No confundir con <del>
66 <meter> Medidor dentro de un rango <meter value="0.7">70%</meter> Diferente de <progress>
67 <noscript> Contenido alterno si no hay JS <noscript>Activa JavaScript</noscript> Soporte para navegadores sin JS
68 <script> Incrusta o enlaza JavaScript <script src="app.js"></script> Atributos: async, defer
69 <style> Define estilos CSS internos <style>body{color:red;}</style> Se coloca en <head>
70 <link> Vincula recursos externos <link rel="stylesheet" href="estilos.css"> Usado para CSS y favicons
71 <base> Define URL base de la página <base href="https://ejemplo.com/"> Afecta enlaces relativos
72 <map> Mapa de imagen <map name="mapa"><area coords="0,0,50,50" href="#"></map> Usado con <img usemap>
73 <area> Zona interactiva en un mapa <area shape="rect" coords="34,44,270,350" href="#"> Depende de <map>
74 <object> Inserta un recurso externo <object data="archivo.pdf" type="application/pdf"></object> Alternativa a <embed>
75 <embed> Contenido incrustado <embed src="animacion.swf"> Usado para multimedia
76 <param> Parámetro para <object> <param name="autoplay" value="true"> Define configuración
77 <track> Subtítulos en video/audio <track src="subs.vtt" kind="subtitles" srclang="es" label="Español"> Funciona dentro de <video> y <audio>
78 <picture> Contenedor de imágenes adaptables <picture><source srcset="img.webp" type="image/webp"><img src="img.jpg"></picture> Optimiza para distintos dispositivos
79 <source> Fuente alternativa para multimedia <source src="video.mp4" type="video/mp4"> Usado en <video>, <audio>, <picture>
80 <bdi> Aísla texto con dirección de escritura <bdi>عربي</bdi> Útil en idiomas RTL
81 <bdo> Sobrescribe la dirección del texto <bdo dir="rtl">Texto invertido</bdo> Soporta valores ltr y rtl
82 <wbr> Posible salto de línea supercalifragilistico<wbr>espialidoso Mejora la legibilidad
83 <template> Fragmento de HTML oculto <template id="tpl"><p>Hola</p></template> Usado con JavaScript
84 <slot> Espacio en Web Components <slot name="contenido"></slot> Se rellena dinámicamente
85 <shadow> Parte del Shadow DOM <shadow></shadow> Elemento experimental
86 <dialog> Ventana emergente modal <dialog open>Hola</dialog> Soporta métodos show() y close()
87 <menu> Lista de comandos o menú <menu><li>Item</li></menu> Actualmente en desuso
88 <menuitem> Elemento de menú contextual <menuitem label="Guardar"> En desuso en navegadores modernos
89 <center> Centra contenido <center>Texto centrado</center> Obsoleto, usar CSS
90 <font> Define fuente, tamaño y color <font color="red">Rojo</font> Obsoleto, reemplazado por CSS
91 <marquee> Texto en movimiento <marquee>Hola</marquee> No estándar, no recomendado
92 <big> Texto más grande <big>Grande</big> Obsoleto, usar CSS
93 <acronym> Define un acrónimo <acronym title="Hypertext Markup Language">HTML</acronym> Obsoleto, usar <abbr>
94 <strike> Texto tachado <strike>Tachado</strike> Obsoleto, usar <s> o <del>
95 <tt> Texto estilo máquina de escribir <tt>Monoespaciado</tt> Obsoleto, usar CSS
96 <plaintext> Muestra todo como texto plano <plaintext>Ignora HTML</plaintext> Obsoleto
100 <ruby> Anotación para caracteres asiáticos <ruby>漢<rt>Kan</rt></ruby> Usado en japonés/chino
98 <rt> Texto de anotación ruby <rt>Kan</rt> Va junto a <ruby>
99 <rp> Texto alternativo para ruby <rp>(</rp><rt>Kan</rt><rp>)</rp> Muestra paréntesis si no hay soporte
100 <rb> Base de anotación ruby <rb>漢</rb> Usado en anotaciones asiáticas
101 <isindex> Campo de entrada único <isindex prompt="Buscar:"> Obsoleto, reemplazado por <input>
102 <frame> Define un marco en frameset <frame src="pagina.html"> Obsoleto en HTML5
